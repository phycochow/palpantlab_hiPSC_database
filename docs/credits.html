Special Acknowledgements

While we have updated the hiPSC-CM protocol database to include more studies and biological annotations, the dataset is originally compiled in a study (in press) by Christopher Chen's lab. The older version of the database is publicly available at https://doi.org//10.5061/dryad.ksn02v7bh.
